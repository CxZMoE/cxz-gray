# cxz-gray
A picture to ascii code generator 一个图片转ASCII文本的小工具

# Usage
```
cxz-gray-ascii.exe

 -i string
        <input file>
  -o string
        <output file> (default "output.png")
  -r int
        <scale ratio> (default 3)
  -s int
        Range:[0-255]
        <alpha channel shreshold filter value.> (default 180)
        
```
